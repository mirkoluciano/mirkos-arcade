# 🕹 Mirko's Arcade

Personal retro gaming station — powered by EmulatorJS.

## Console supportate
- Nintendo NES
- Super Nintendo (SNES)
- Game Boy / Game Boy Color / Game Boy Advance
- Sega Mega Drive
- Sega Master System / Game Gear
- Sony PlayStation 1
- Nintendo 64
- Nintendo DS
- Arcade / MAME

## Come si usa
1. Scegli la console
2. Clicca "CARICA ROM" e seleziona il file ROM dal tuo dispositivo
3. Gioca!

Le ROM non vengono mai caricate sul server — restano sempre sul tuo dispositivo.

## Setup GitHub Pages
1. Crea repo su GitHub
2. Carica tutti i file (index.html, manifest.json, sw.js)
3. Vai su Settings → Pages → Branch: main → Save
4. Il sito sarà disponibile su `https://mirkoluciano.github.io/<nome-repo>/`

## Icone PWA
Crea una cartella `icons/` con:
- `icon-192.png` (192×192px)
- `icon-512.png` (512×512px)

Puoi usare qualsiasi immagine 8-bit / pixel art come icona!
